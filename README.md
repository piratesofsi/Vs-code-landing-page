


# VS Code Landing Page Clone

A Visual Studio Code inspired landing page built using HTML and CSS only.  
This project was created as a UI practice exercise to recreate the design, layout style, and visual elements of the official VS Code website using only fundamental web technologies.

---

## Features

- Navigation bar with hover effects
- Gradient-based hero section
- Call-to-action button with smooth transitions
- Language icons section using CSS Grid
- Clean visual spacing and modern typography
- Smooth hover animations and color transitions
- Dark UI theme similar to VS Code

---

## Tech Stack

| Technology | Purpose |
|-----------|----------|
| HTML5 | Structure and content |
| CSS3 | Styling, layout, transitions, animations |
